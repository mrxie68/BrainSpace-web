BrainSpace基本使用文档

2024年11月17日 06:33 小茗同学 本文共：97 字 阅读需要：1 分钟 加载中...

安装使用

确保已经正确安装 Hugo（具体安装步骤请参考 Hugo 官方文档）。在终端中新建一个文件夹作为博客项目目录，并进入该目录。然后运行以下命令以生成基本目录结构，下面将以myblog演示：

hugo new site . --force

上述命令会在当前路径中创建 Hugo 的默认目录结构。

将主题 BrainSpace 的文件夹拷贝到 myblog/themes 文件夹下。然后将主题文件夹中的 theme.toml 配置文件内容复制到博客根目录的 hugo.toml 文件中，并移除其中的所有注释。

预览

hugo server

启动成功后，Hugo 会提供一个本地地址（如 http://localhost:1313），通过浏览器访问即可预览博客内容。

创建主要目录结构

在content文件夹新建class、post文件夹和_index.md。class将用来存储单独的页面（如 about.md、message.md 等），post用于存储博客文章文件。

完成以上步骤后，博客的主要目录结构应如下：

myblog/
├── content/
│   ├── class/
│   │   └── about.md
│   ├── post/
│   │   └── test.md
│   │── _index.md

注意：post文件夹里面的markdown数量要大于class文件夹的数量，否则可能会造无法正确显示的情况。这是一个bug,我目前还没有找到修复的办法。 目前的解决办法是：可以在post多复制几个在文章头部信息加上draft: true。

文章头部信息说明

字段名描述示例
title文章标题，显示在页面顶部和浏览器标题栏中。"BrainSpace基本使用文档"
date发布时间，格式为 ISO8601 标准。2024-11-17T06:33:00+08:00
author作者的姓名或笔名，通常显示在文章顶部或元数据区域。"小茗同学"
tags文章分类或标签，此主题已被修改成分类。["博客搭建"]
weight权重，用于控制文章在列表中的排序，数值越小优先级越高。1
blogTitle自定义文章页面底部版权信息。© 2024 小茗同学的博客
enableComments是否启用评论功能，设置为 true 时开启，false 时关闭。默认 false，仅支持
featured_image文章特色图片路径，用于封面、列表页或社交媒体分享。"/2024/11/cover.jpg"
summary文章摘要，用于概述内容，通常显示在文章列表页或摘要视图中。"本教程介绍 Hugo 的安装与配置方法。"
